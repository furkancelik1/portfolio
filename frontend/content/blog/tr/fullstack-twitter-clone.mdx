## Proje Hakkında

Full-stack bir Twitter klonu geliştirerek **MERN Stack**'teki yetkinliğimi ileri seviyeye taşıdım.

## Kullanıcı Kimlik Doğrulama

JWT tabanlı auth sistemi şu şekilde çalışıyor:

```javascript
const generateToken = (user) => {
  return jwt.sign(
    { id: user._id, email: user.email },
    process.env.JWT_SECRET,
    { expiresIn: "7d" }
  );
};
```

## Gerçek Zamanlı Mesajlaşma

Socket.io ile anlık mesajlaşma ve bildirim sistemi kurdum:

```javascript
io.on("connection", (socket) => {
  socket.on("join", (room) => {
    socket.join(room);
  });

  socket.on("send_message", async (data) => {
    const message = await Message.create(data);
    io.to(data.room).emit("new_message", message);
  });
});
```
