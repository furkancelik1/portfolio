## Staj Süreci

Mozaik Bilişim'deki stajım boyunca frontend ve backend geliştirme süreçlerinde aktif rol aldım.

## Edindiğim Deneyimler

- **Kod Review Süreçleri:** Takım arkadaşlarımın kodlarını incelemek bana farklı bakış açıları kazandırdı.
- **Production Hataları:** Canlı sistemde karşılaşılan hataları debugging etmek, bootcamp'te öğrenilenden çok farklıydı.
- **Ekip İletişimi:** Günlük stand-up'lar ve sprint planlamaları ile Agile metodolojisini deneyimledim.

## Teknik Kazanımlar

Staj süresince düzenli olarak kod yazdığım teknolojiler:

```
- React ile component geliştirme
- Node.js / Express backend servisleri
- PostgreSQL veritabanı sorguları
- Git flow ve pull request yönetimi
```

## Tavsiyeler

Staja başlayacak arkadaşlara en büyük tavsiyem: **bol bol soru sormaktan çekinmeyin.**
